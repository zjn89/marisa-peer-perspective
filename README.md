# Marisa Peer Perspective

Marisa Peer 思维框架 Skill——由 [女娲 · Skill造人机](https://github.com/alchaincyf/nuwa-skill) 生成。

## 这是什么

这是一个 AI Skill，蒸馏了英国心理治疗师、RTT（Rapid Transformational Therapy）创始人 **Marisa Peer** 的思维框架。

包含：
- **6 个心智模型**：我不够好根因模型、心智服从文字与画面、熟悉感驱动一切、"更好的谎言"叙事重构、语言即指令（DICCC）、童年编程（RFPI）
- **10 条决策启发式**
- **完整表达 DNA**：句式、隐喻、高频用语、频率约束
- **Agentic Protocol**：多步骤回答工作流 + 高风险决策安全护栏
- **诚实边界**：5 项局限、6 处矛盾、监管裁决记录

## 调研来源

12 个一手/二手英文来源，包括 IPSO 裁决、ASA 裁定、27 场播客访谈、9 本书、NHS/DHSC 公开声明等。

## 使用方式

将 `SKILL.md` 放入 `.claude/skills/marisa-peer-perspective/` 目录即可激活。

触发词：Marisa Peer、RTT、快速转化疗法、我不够好、更好的谎言、催眠治疗、心智法则、I am enough
